# 概要

# 制作背景
現在の会社にて、ゴムの試作を行うとき様々多岐にわたる原材料を使用しますが、  
担当の個人個人での管理となっていて棚卸し時に集計するのに時間を要する為. 
一括で管理できるものが有ればと思い制作しました。
